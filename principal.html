OBJETIVOS

Aminorar desabasto

Con el programa cosecha de lluvia se busca mejorar el abastecimiento y recolección de la precipitación pluvial, es un proyecto para aprovechar el agua que cae del cielo. Se usan techos y tubos para juntar el agua, la filtra y la guarda en tanques. Esto ayuda a tener agua extra en casa para usar en lo que necesites, especialmente cuando falta el vital líquido de otras fuentes.

Cosecha de lluvia

La meta es que todos los habitantes de la comunidad sean beneficiados en recibir el agua, a través de instalar sistemas de recolección de agua de lluvia, en hogares y edificios públicos, aumentando la disponibilidad de agua y reduciendo la dependencia de fuentes externas durante temporadas de sequías.

Uso de tecnologias inteligentes

Fomentar el uso de tecnologías como regaderas y llaves inteligentes, para reducir el consumo de agua en hogares y empresas, permitiendo su reutilización en actividades no potables y ayudando a conservar el recurso hídrico.

INSTALACIÓN

Un Sistema de Captación de Agua de Lluvia (SCA) recoge, almacena y trata el agua mediante canaletas, filtros, bombas y tanques. Para la instalación, se calcula el área del techo y las precipitaciones locales. Es clave desviar la "primera lluvia", que arrastra suciedad, y filtrar residuos antes del tanque. El tanque debe estar cerrado para evitar luz, algas y animales. Incluye un mecanismo para desviar agua no deseada y mantener el sistema limpio y eficiente

Un diagrama de flujo es una representación gráfica y secuencial de un proceso o flujo de trabajo con todas las tareas y actividades principales necesarias para lograr un objetivo común.

El diagrama de Gantt es una forma de visualizar el programa del proyecto y darle seguimiento a los logros y de estar siempre familiarizado con el cronograma del trabajo. Cada barra de un diagrama de gantt representa una etapa del proceso y su longitud y la duración del proyecto.


Este organigrama muestra cómo se organiza el equipo de un proyecto de sostenibilidad hídrica, es decir, un plan para cuidar y usar el agua de manera responsable. Cada persona tiene un papel importante para que todo funcione bien:
Martha, la directora general, es la líder del equipo. Ella toma las decisiones más importantes, analiza la información que se va a usar y se encarga de que se impriman materiales informativos, como trípticos y lonas, para enseñar a las personas cómo recolectar agua de lluvia.
Nicolás Velasco, el subdirector, es el brazo derecho de Martha. Él organiza al equipo y los guía para que todos sepan qué hacer. También asegura que lo que el equipo diga sea claro, fácil de entender y confiable.
Moisés, el auxiliar de oficina, hace trabajos clave como organizar documentos, guardar información y asegurarse de que todo lo planeado en el proyecto suceda como se espera.
Olga Lidia, la ingeniera, es la encargada de las soluciones técnicas. Ella diseña procesos para captar, tratar y usar el agua de manera eficiente. También supervisa que todo lo técnico se haga bien y funcione correctamente.
Armando, el proveedor de materiales, se encarga de traer todo lo necesario para construir o instalar lo que se necesita en el proyecto, como los sistemas de recolección de agua. Leticia, la asesora de materiales compatibles, ayuda a elegir los mejores materiales que funcionen bien juntos, que sean seguros y que duren mucho tiempo.
